## Установка компилятора

### 1. На Fedora / Red Hat / CentOS
Обновите списки пакетов и установите актуальную версию `g++` :
```bash
sudo dnf install gcc-c++
```

### 2. На Ubuntu / Debian / Linux Mint
Обновите репозитории и установите компилятор `g++`:
```bash
sudo apt update
sudo apt install g++
```

### 3. На Arch Linux / Manjaro
Установите компилятор `g++`:
```bash
sudo pacman -Syu gcc
```


###  Как проверить версию компилятора перед сборкой?
Перед компиляцией убедитесь, что версия вашего `g++` не ниже 7.0. Для этого выполните команду:
```bash
g++ --version
```
Если в первой строчке вывода указана версия, например, `g++ (GCC) 11.2.1` или выше — ваш компилятор полностью готов к работе со стандартом **C++17** .

## Установка библиотеки Boost.asio (версии 1.70 или новее)

### На Fedora / Red Hat / CentOS:
  ```bash
  sudo dnf install boost-devel
  ```

### На Ubuntu / Debian / Linux Mint:
  ```bash
  sudo apt update
  sudo apt install libboost-all-dev
  ```
  *(Или точечно `sudo apt install libboost-system-dev`, но `libboost-all-dev` гарантирует, что подтянутся все зависимости Asio).*

### На Arch Linux / Manjaro:
  ```bash
  sudo pacman -Syu boost
  ```

## Компиляция Клиента и Сервера (на все дистрибутивы)

### Компиляция Сервера 
```bash
  g++ -std=c++17 -I. Session/SessionServer.cpp Server/Server.cpp mainServer.cpp -o server -lboost_system 
  ```

### Компиляция Клиента
```bash
  g++ -std=c++17 -I.  Client/Client.cpp mainClient.cpp -o client -lboost_system
  ```
